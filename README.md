# Food Nutrition Analyzer 🍎📊

A robust Python application designed to analyze, process, and extract actionable insights from food nutritional data. This tool parses a comprehensive dataset of various food items to provide key nutritional metrics, making it easier to track macros, calories, and overall dietary information.

## 📁 Repository Structure

* **`app.py`**: The main application script that runs the analyzer.
* **`nutrients_data.csv`**: The core dataset containing various food items and their detailed nutritional values.
* **`requirements.txt`**: The list of Python dependencies required to run the project.
* **`.devcontainer/`**: Configuration files for a seamless containerized development environment (ideal for GitHub Codespaces or VS Code Dev Containers).

## 🚀 Getting Started

### Prerequisites
Ensure you have Python 3.7 or higher installed on your system.

### Installation

1. Clone the repository to your local machine:
   ```bash
   git clone [https://github.com/Lalithsai0306/food-nutrition-analyzer-.git](https://github.com/Lalithsai0306/food-nutrition-analyzer-.git)
   cd food-nutrition-analyzer-
   ```

2. Create and activate a virtual environment (recommended):
   ```bash
   # On macOS/Linux
   python -m venv venv
   source venv/bin/activate
   
   # On Windows
   python -m venv venv
   venv\Scripts\activate
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage

Run the main application script from your terminal:
```bash
python app.py
```
*(Note: If you built this user interface using a web framework like Streamlit, run `streamlit run app.py` instead.)*
